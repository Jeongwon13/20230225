변수 선언 방식

        Javascript는 변수 선언 시 자로형이 아닌
        var, let, const, (미작성) 중 하나를 골라 작성함

        var : 변수, 변수명 중복 가능(덮어쓰기 가능), 함수 레벨 scope
        let : 변수, 변수명 중복 X, 블록{} 레벨 scope
        const : 상수, 변수명 중복 X, 블록{} 레벨 scope

        1순위: const
            - JS는 HTML상에서 특정 요소를 선택해 지속적으로 추적하여 사용하는 경우가 많음.
            - 하여, 상수명 변수에 저장해 고정적으로 사용
        2순위: let
        3순위: var
        비추천: (미작성)

JavaScript의 자료형

        JS는 변수 선언 시 별도의 자료형을 지정하지 않는다.
        -> 변수에 대입되는 값(리터럴)에 의해 자료형이 결정된다.

        - string(문자열 / "abc", 'abc')
        - number(숫자 / 100, -3.14)
        - boolean(논리값 / true, false)
        - object(객체 / 배열(Array), JS객체{Key:Value, K:V}) 
        - function(함수)
        - undefined(자료형이 정의되지 않은 변수) -> 값이 대입되지 않은 변수
        
        - null (참조하지 않음. *자료형이 아니다!!!)
